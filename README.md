# 🩸 **Red Cross: Blood+**
### *An Integrated Blood Donation Management System for the Philippine Red Cross, Camarines Norte Chapter*

---

## 📖 **Project Description**
> **Red Cross: Blood+** is a centralized **web and mobile application** developed for the  
> **Philippine Red Cross – Camarines Norte Chapter**.  
> It streamlines the management of **blood donation drives**, **donor registration**, and **blood bank inventory monitoring**.  

💡 *The system bridges aspiring and repeat donors with Red Cross administrators, providing real-time updates on upcoming donation events and blood availability. It also encourages public engagement through donor profiles, facts, and nearby blood bank information.*

---

## 🚀 **Core Features**

### 🌐 **Web Application (Admin Panel)**
> 🧰 Designed for Red Cross administrators to manage drives, inventory, and donor data.

#### 🩸 Blood Drive Posting & Announcements
- ➕ Add, ✏️ edit, or ❌ delete blood donation drive posts.  
- 📊 Track donor registration per drive.  
- 💉 Monitor total blood volume collected.

#### 🧪 Blood Bank Inventory Management
- 🩸 View available blood volume per blood type.  
- 📈 Generate summary and analytics reports.

#### 👥 Registered Donors Listing
| Field | Description |
|-------|--------------|
| 🧑 Name | Full name of the donor |
| 🎂 Age | Donor’s age |
| 🚻 Sex | Gender identity |
| 🩸 Blood Type | e.g., A+, O−, B+, AB+ |
| ⚖️ Height & Weight | Used for eligibility assessment |
| ❤️ Comorbidities | Health conditions declaration |

---

### 📱 **Mobile Application (Donor App)**
> 📲 Built for aspiring and repeat donors to manage their profiles, locate blood drives, and learn about donation benefits.

#### 🧾 Donor Registration
- Fill in required health and contact details.

#### 👤 Donor Profile Management
- Update medical and personal information every six (6) months.

#### 🏥 Blood Banks Near Me
- Locate nearby Red Cross blood banks within **Camarines Norte**.

#### 📅 Blood Drives Near Me
- View ongoing or upcoming donation drives by Red Cross or partner NGOs.

#### 🧠 Educational Corner
- Access facts, trivia, and health benefits about blood donation.

## ⚙️ **Installation Guide**
```bash
### 🖥️ Web Application

# 1️⃣ Clone the repository
git clone https://github.com/your-org/redcross-bloodplus.git

# 2️⃣ Navigate into the project folder
cd redcross-bloodplus/web

# 3️⃣ Install dependencies
npm install

# 4️⃣ Run the development server
npm start

### 📱 Mobile Application

# 1️⃣ Navigate to the mobile folder
cd redcross-bloodplus/mobile

# 2️⃣ Install dependencies
npm install

# 3️⃣ Run the app
npm start

```
